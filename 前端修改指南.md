## 修改网站标题

网站标题显示在浏览器标签页上。

### 修改位置

文件：`frontend/src/components/Layout.tsx`

### 查找并修改

1. 用编辑器打开 `frontend/src/components/Layout.tsx`
2. 找到这一行（大约在第 30-40 行）：
   ```typescript
   document.title = "羚羊日寄(๑˃̵ᴗ˂̵) ♡";
   ```
3. 修改为你想要的标题，例如：
   ```typescript
   document.title = "我的日记本";
   ```
4. 保存文件

### 修改离开标签页时的标题

同样在这个文件中，找到：
```typescript
document.title = "你别走呀˚‧º·(˚ ˃̣̣̥᷄⌓˂̣̣̥᷅ )‧º·˚";
```
修改为你想要的文字。

## 修改颜色主题

### 修改位置

文件：`frontend/src/index.css`

### 主要颜色说明

在 `index.css` 文件中，你可以找到这些颜色定义：

```css
/* 背景颜色 */
background: linear-gradient(to bottom, #f0f4f8, #e2e8f0);

/* 主色调（蓝色系）*/
--primary-color: #4f46e5;  /* 主色 */
--secondary-color: #6366f1; /* 次色 */

/* 文字颜色 */
--text-primary: #1e293b;   /* 主要文字 */
--text-secondary: #64748b; /* 次要文字 */
```

### 修改步骤

1. 打开 `frontend/src/index.css`
2. 找到颜色定义（通常在文件开头）
3. 修改颜色值

**颜色值格式：**
- 使用十六进制：`#4f46e5`（蓝色）
- 使用 RGB：`rgb(79, 70, 229)`

**常用颜色参考：**
- 蓝色：`#3b82f6`、`#2563eb`
- 绿色：`#10b981`、`#059669`
- 紫色：`#8b5cf6`、`#7c3aed`
- 粉色：`#ec4899`、`#db2777`
- 橙色：`#f59e0b`、`#d97706`

### 修改背景颜色

找到这一行：
```css
background: linear-gradient(to bottom, #f0f4f8, #e2e8f0);
```

修改为你喜欢的颜色，例如：
```css
/* 浅蓝色背景 */
background: linear-gradient(to bottom, #e0f2fe, #bae6fd);

/* 浅粉色背景 */
background: linear-gradient(to bottom, #fdf2f8, #fce7f3);

/* 纯色背景 */
background: #ffffff;
```

## 修改字体

### 修改位置

文件：`frontend/src/index.css`

### 当前使用的字体

项目使用了多种字体：
- **标题字体**：`Noto Serif SC`（衬线字体）
- **装饰字体**：`Ma Shan Zheng`（毛笔字体）
- **正文字体**：系统默认字体

### 修改标题字体

1. 打开 `frontend/src/index.css`
2. 找到字体导入部分（文件开头）：
   ```css
   @import url('https://fonts.googleapis.com/css2?family=Noto+Serif+SC:wght@400;700&display=swap');
   ```
3. 可以替换为其他 Google 字体，例如：
   ```css
   /* 使用思源黑体 */
   @import url('https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;700&display=swap');
   
   /* 使用微软雅黑风格 */
   @import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap');
   ```
4. 找到字体定义部分：
   ```css
   .font-title {
     font-family: 'Noto Serif SC', serif;
   }
   ```
5. 修改为对应的字体名称

### 添加新字体

1. 在 `index.css` 开头添加字体导入：
   ```css
   @import url('https://fonts.googleapis.com/css2?family=你的字体名称&display=swap');
   ```
2. 在字体定义部分添加：
   ```css
   .font-title {
     font-family: '你的字体名称', serif;
   }
   ```

**推荐字体：**
- 中文：`Noto Sans SC`、`Source Han Sans SC`
- 英文：`Inter`、`Roboto`、`Open Sans`

## 修改时间显示格式

### 修改位置

文件：`frontend/src/pages/Home.tsx`

### 查找时间格式化函数

1. 打开 `frontend/src/pages/Home.tsx`
2. 搜索 `formatDate` 或 `formatTime`
3. 找到类似这样的代码：
   ```typescript
   const formatDate = (dateString: string) => {
     const date = new Date(dateString);
     return date.toLocaleDateString('zh-CN', {
       year: 'numeric',
       month: 'long',
       day: 'numeric',
       timeZone: 'Asia/Shanghai'
     });
   };
   ```

### 修改日期格式

修改 `toLocaleDateString` 的选项：

```typescript
// 只显示年月
return date.toLocaleDateString('zh-CN', {
  year: 'numeric',
  month: 'long',
  timeZone: 'Asia/Shanghai'
});

// 显示完整日期（年月日）
return date.toLocaleDateString('zh-CN', {
  year: 'numeric',
  month: '2-digit',
  day: '2-digit',
  timeZone: 'Asia/Shanghai'
});

// 使用斜杠分隔
return date.toLocaleDateString('zh-CN', {
  year: 'numeric',
  month: '2-digit',
  day: '2-digit',
  timeZone: 'Asia/Shanghai'
}).replace(/\//g, '-');  // 将 / 替换为 -
```

### 修改时间格式

找到 `formatTime` 函数，修改：

```typescript
// 12小时制
return date.toLocaleTimeString('zh-CN', {
  hour: '2-digit',
  minute: '2-digit',
  hour12: true,  // 使用 12 小时制
  timeZone: 'Asia/Shanghai'
});

// 24小时制（默认）
return date.toLocaleTimeString('zh-CN', {
  hour: '2-digit',
  minute: '2-digit',
  hour12: false,  // 使用 24 小时制
  timeZone: 'Asia/Shanghai'
});
```

## 修改布局间距

### 修改位置

文件：`frontend/src/pages/Home.tsx` 和 `frontend/src/components/Layout.tsx`

### 修改内容区域宽度

1. 打开 `frontend/src/components/Layout.tsx`
2. 找到 `max-w-5xl` 或类似的类名
3. 修改为：
   - `max-w-4xl` - 更窄
   - `max-w-5xl` - 中等（当前）
   - `max-w-6xl` - 更宽
   - `max-w-7xl` - 很宽

### 修改时间轴位置

1. 打开 `frontend/src/pages/Home.tsx`
2. 搜索 `left-` 或 `ml-`（左边距）
3. 找到时间文本的位置，例如：
   ```tsx
   <div className="left-8 ...">
   ```
4. 修改数字：
   - `left-4` - 更靠右
   - `left-8` - 中等
   - `left-12` - 更靠左
   - `left-16` - 很靠左

## 应用修改

### 本地测试

修改后，需要在本地测试：

1. **停止当前服务**（如果正在运行）
   ```bash
   docker-compose down
   ```

2. **重新构建前端**
   ```bash
   docker-compose up -d --build frontend
   ```

3. **查看效果**
   - 访问 `http://localhost:3000`
   - 检查修改是否正确

### 部署到服务器

修改完成后，需要重新部署：

1. **提交代码**（如果使用 Git）
   ```bash
   git add .
   git commit -m "修改前端样式"
   git push
   ```

2. **在服务器上更新**
   ```bash
   # 拉取最新代码
   git pull
   
   # 重新构建并启动
   docker-compose -f docker-compose.prod.yml up -d --build
   ```
