# 快速启动指南

## 前提条件

1. ✅ 已安装 Docker Desktop（版本 26.1.3 或更高）
2. ✅ Docker Desktop 正在运行（系统托盘图标显示运行中）

## 启动步骤

### 方法一：使用启动脚本（推荐）

**Windows:**
```bash
start.bat
```

**Linux/Mac:**
```bash
chmod +x start.sh
./start.sh
```

### 方法二：手动启动

1. **确保 Docker Desktop 正在运行**
   - 打开 Docker Desktop 应用
   - 等待完全启动（状态栏显示 "Docker Desktop is running"）

2. **构建并启动服务**
   ```bash
   docker-compose up -d --build



3. **查看服务状态**
   ```bash
   docker-compose ps
   ```

4. **查看日志（如果需要）**
   ```bash
   docker-compose logs -f
   ```

## 访问应用

启动成功后，在浏览器中访问：

- **前端首页：** http://localhost:3000
- **管理页面：** http://localhost:3000/admin
- **后端 API：** http://localhost:3001/api/health

## 停止服务

```bash
docker-compose down
```

## 常见问题

### Q: 提示 "Docker Desktop is not running"
**A:** 请先启动 Docker Desktop 应用程序，等待其完全启动后再运行启动脚本。

### Q: 端口被占用
**A:** 修改 `docker-compose.yml` 中的端口映射，或关闭占用端口的程序。

### Q: 构建失败
**A:** 检查网络连接，确保能正常访问 npm registry。

### Q: 无法访问页面
**A:** 
1. 检查 Docker Desktop 是否运行
2. 运行 `docker-compose ps` 查看容器状态
3. 运行 `docker-compose logs` 查看错误日志

更多问题请参考 `故障排除.md`
